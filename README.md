# 💄 Makeup Sales Data Analysis (2025)

## 📌 Project Overview

This project analyzes a makeup sales dataset to uncover key business insights related to revenue, customer behavior, and product performance.

---

## 📊 Dataset Features

* Sale_ID
* Date
* Brand
* Product_Type
* Country
* Sales_Channel
* Payment_Method
* Price_USD
* Units_Sold
* Revenue_USD

---

## 🎯 Objectives

* Analyze revenue trends over time
* Identify top-performing brands and products
* Understand customer purchasing behavior
* Evaluate sales channels and payment methods
* Derive actionable business insights

---

## 🧹 Data Processing

* Extracted Month and Day features
* Checked for missing values (none found)
* Checked for duplicated values (none found)
* Converted Date column to datetime
---

## 📈 Key Analyses Performed

* Monthly revenue trend analysis
* Brand-wise revenue comparison
* Product category contribution
* Country-wise sales performance
* Sales channel and payment method analysis
* Correlation analysis between price, units sold, and revenue
* KPI analysis (Total Revenue, Avg Order Value, Units Sold)

---

## 📊 Dashboard

A comprehensive dashboard was created using multiple visualizations including:

* Line charts
* Bar charts
* Pie charts
* Scatter plots
* Heatmaps
* KPI visualizations

---

## 🧠 Key Insight

**Revenue is primarily driven by high sales volume, with variations across brands, products, and regions, indicating strong demand and opportunities to improve average order value.**
